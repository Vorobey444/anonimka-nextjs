2.1. **Характер Сервиса**  
Anonimka представляет собой онлайн-платформу для социального взаимодействия и знакомств совершеннолетних пользователей, предоставляющую технические средства для анонимной коммуникации. 2.2. **Функциональные возможности**  
Приложение предоставляет следующие функции:
- Создание и размещение пользовательских анкет (профилей) для знакомств
- Система приватного обмена сообщениями между пользователями
- Общедоступный чат для группового общения ("Мир чат")
- Просмотр и поиск анкет других пользователей по различным критериям
- Система геолокации для поиска пользователей в определенной местности

2.3. **Ограничения Сервиса**  
Администрация НЕ предоставляет:

- Модерацию контента в режиме реального времени
- Гарантии безопасности встреч в реальной жизни

2.4. **Целевая аудитория**  
Сервис предназначен исключительно для совершеннолетних пользователей (18+) и ориентирован на аудиторию, проживающую на территории Республики Казахстан, Российской Федерации и стран СНГ.
