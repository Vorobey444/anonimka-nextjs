# Применение миграции 021

## Что делает миграция
Создаёт таблицы для хранения истории чатов (общего и приватных) для системы модерации.

## Таблицы
- `world_chat_messages` - история общего чата "Мир чата"
- `private_chat_messages` - история приватных чатов

## Как применить

### Через Neon Console (рекомендуется)
1. Открой https://console.neon.tech/
2. Выбери свой проект `anonimka`
3. Перейди в SQL Editor
4. Скопируй и выполни содержимое файла `021_add_chat_history.sql`

### Проверка
```sql
SELECT table_name 
FROM information_schema.tables 
WHERE table_name IN ('world_chat_messages', 'private_chat_messages');
```

Должно вернуть обе таблицы.
